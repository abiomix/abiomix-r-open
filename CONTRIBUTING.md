# Contributing

Contributions should solve a generally applicable package or upstream-tool
problem. Open an issue with a minimal failing fixture before a substantial
patch.

For imported upstream code, state:

- upstream repository and exact commit;
- package and bundled-component licenses;
- the behavior being changed;
- why the change cannot yet be consumed upstream;
- the tests that distinguish the patched behavior;
- the plan for upstreaming and synchronizing the patch.

Do not submit product rules, private data, credentials, patient identifiers, or
customer-specific report content.

All Abiomix-maintained packages follow the repository `Authors@R` policy:
Sounkou Mahamane Toure is an author, and Abiomix FZ LLC is recorded as copyright
holder and funder for Abiomix contributions. Upstream attribution is additive
and must never be removed.
